# Voice Command Automation using Python

## Description
This project is a simple voice-based system that performs tasks using voice commands.

##  Features
- Opens YouTube, Google
- Opens Notepad
- Opens Camera and WhatsApp
- Tells time and date
- Uses speech recognition and text-to-speech

##  Technologies Used
- Python
- SpeechRecognition
- pyttsx3
- Webbrowser
- OS module

##  How to Run
1. Install required libraries:
   pip install SpeechRecognition pyttsx3 pyaudio

2. Run the notebook
